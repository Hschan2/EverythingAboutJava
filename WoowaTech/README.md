# 우아한테크 수강생들 발표 웹 지식
우아한테크 수강생들이 학습하고 발표한 지식들을 기록합니다. 그들이 학습한 내용들을 학습해 동시에 지식을 늘리고 몰랐던 부분은 학습하고 이해하려고 노력합니다. 그리고 그들이 전달하는 팁들을 확인하고 학습합니다.   

수강생들이 학습을 위한 레퍼런스를 참고하고 이 후 지식 향상을 위해 해당 레퍼런스들을 참고하여 학습합니다.
